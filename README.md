# 3D Multivariable Function & Gradient Visualizer

The **3D Multivariable Function & Gradient Visualizer** is a Python-based project developed to help students understand important concepts in **Multivariable Differential Calculus** through graphical visualization.

The project plots **three-dimensional surfaces of functions of two variables** and displays **gradient vectors** that represent the direction of the maximum rate of increase of the function. This tool bridges the gap between **theoretical mathematics and practical visualization** using computational methods.

---

## Features

* Visualization of **3D surfaces of multivariable functions**
* Computation of **partial derivatives**
* Visualization of **gradient vectors**
* Interactive **3D plot (rotate and zoom)**

## Mathematical Concept

The project visualizes functions of the form:

z = f(x, y)

Example function used in the project:

f(x, y) = x² + y²

Partial derivatives:

∂f/∂x = 2x
∂f/∂y = 2y

Gradient vector:

∇f = (fx, fy)

The **gradient vector** indicates the direction of the **maximum rate of increase of the function**.

## Algorithm

1. Import required Python libraries (NumPy, Matplotlib)
2. Define the mathematical function
3. Define the partial derivatives
4. Create a Cartesian grid using `meshgrid`
5. Compute function values on the grid
6. Calculate gradient components
7. Plot the 3D surface
8. Display gradient vectors
9. Show the final visualization
